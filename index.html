Web 101

Class 1

In this class you will learn the fundamentals of designing and building a web page. You will learn the primary technologies involved in building web pages (HTML, CSS, and Javascript). We will also introduce your project.

Topics Covered

  • Intro to Web
  • The DOM - The Basic HTML Tags
  • Rendering
  • Linking JavaScript and CSS - What is the difference between Inline and Linked CSS?
  • Separation of Concerns - Content vs Presentation vs Functionality

Students should...

  • Understand what goes on behind the scenes on a web page
  • Know how to write basic HTML
  • Know the purpose of HTML, JS, and CSS in a webpage

Class 2

The second class is devoted to CSS. We will teach you how to use it and what the most-used CSS properties do with the goal of building a two-column layout.

Topics Covered

  • CSS Syntax
  • CSS Selectors
  • The IMG Tag
  • Inline vs. Inline Block Display
  • A Simple CSS layout

Students should...

  • Feel comfortable using CSS and looking up additional properties online
  • Appreciate the inline-block value for the display
  • Be able to build a CSS layout at home

Class 3

Now that you have the knowledge of how to build a web page, you will learn how to leverage some helpful and free tools like Chrome's Developer Tools. You will also learn how to get your code online by pushing to Github.

Topics Covered

  • Browser Dev Tools
  • How to use Git with Eclipse - Pushing and Pulling
  • Intro to JS - document.getElementById

Students should...

  • Be able to perform basic operations in the tools we show you
  • Push your HTML/CSS files to your github repository

Class 4

In this class you will review everything you learned in the class and have the opportunity to ask the instructor questions. After that, students will present their website pages to the class.

Topics Covered

  • Review

Students should...

  • Have created their own webpage
  • Be able to push and pull with Github
  • Know where to find any additional information they need so they can learn how to build more complex websites.
